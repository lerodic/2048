- Merge tiles with the same number to create bigger numbers.
- Try to get as high of a score as you can.

## Installation

```bash
# Clone the repository
git clone https://github.com/lerodic/2048.git

# Navigate into the project
cd 2048

# Install dependencies
npm install

# Build the project
npm run build

# Preview
npm run preview

# Open your browser and navigate to the local URL (http://localhost:4173/ by default)
```
